# Iconos — placeholders

Reemplazar `icono1.svg`, `icono2.svg`, ... por los archivos finales de la diseñadora,
manteniendo los mismos nombres para no romper imports.

Convención sugerida:
- icono1.svg → flecha izquierda (carruseles)
- icono2.svg → flecha derecha (carruseles)
- icono3.svg → logo / monograma "Artes Lodge"
- icono4.svg → ícono de ubicación / pin
- icono5.svg → ícono de contacto / mail
- ...
